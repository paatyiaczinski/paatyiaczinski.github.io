Análise de Marketing

🧩 Contexto do Projeto

Análise interativa de dados de marketing (simulados) focada em comportamento do cliente, desempenho de campanhas (ROI/CAC) e padrões de compra. O projeto aplica técnicas de segmentação RFM e visualização em Power BI para fornecer insights acionáveis a gestores de marketing.

📝 Destaques
  • Segmentação RFM com pontuação e classificação personalizada
  • Métricas como ROI, CAC, Ticket Médio e Conversão por campanha
  • Comparações entre canais de venda e comportamento por país
  • Uso de filtros avançados e painel retrátil para navegação
🛠️ Ferramentas Utilizadas
  • Power BI – Modelagem de dados, fórmulas DAX e visualizações interativas.
  • Excel/CSV - Manipulação e limpeza de dados
  • GitHub Projects - Gerenciamento e documentação de tarefas
